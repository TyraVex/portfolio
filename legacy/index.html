# Hello !

I am TyraVex, a self-taught developper and an engineering lover.

I love code, cybersecurity, mechatronics and managment.

Automating tasks, playing with APIs, AIs, and new technologies are my computer related passions.

Realising new projects using electronics, soldering, a modelisation software and my 3D printer make me enjoy the fascinating world of ingeneering.

# About this page

This page is still in development. It is planned to be merged with the mpp.pw main page.

I made this website to showcase my mechatronics, coding and cybersecurity projects.

I also wanted to a create a place to express who I am and what I truly love.

Finally, I was attracted by web technologies, and I am glad that I now learned more about HTML, CSS, and Javascript.

In fact, making this website entirely by myself (exept the three.js framework used on the main page for 3D rendering) allowed me understand how these languages work in a deeper way.
