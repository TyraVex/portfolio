# Hello !

I am TyraVex, a self-taught developer and an engineering lover.

I am enthralled by programming, cybersecurity, mechatronics and management.

Automating tasks, playing with APIs, AIs, and new technologies are my computer related passions.

Realizing new projects using electronics, soldering, a modeling software and my 3D printer make me enjoy the fascinating world of engeneering.

In a few worlds, building the software of tomorrow, performing penetration tests and combining these with electronics in any way is my ultimate dream.

# About this page

This page is still in development. It is planned to be merged with the mpp.pw main page.

I made this website to showcase my mechatronics, coding and cybersecurity projects.

I also wanted to create a place to express who I am and what I truly love.

Finally, I was attracted by web technologies, and I am glad that I now learned more about HTML, CSS, and Javascript.

In fact, making this website entirely by myself (except the three.js framework used on the main page for 3D rendering) allowed me to understand how these languages work in a deeper way.
